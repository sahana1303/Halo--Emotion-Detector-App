# HALO Emotion Detector App

## 📌 Overview
This project is an AI-based Emotion Detection system that identifies human emotions from input data.

## 🚀 Features
- Detects emotions using Machine Learning
- User-friendly web interface
- Real-time predictions

## 🛠️ Tech Stack
- Python
- Flask
- HTML/CSS
- Machine Learning

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py

## 📊 Output
The system predicts emotions such as Happy, Sad, Angry, etc.
